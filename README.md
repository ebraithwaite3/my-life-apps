# My Life Apps

Monorepo containing all my life management apps.

## Apps
- **Workout App**: Track workouts and fitness
- **Calendar App**: Calendar and scheduling

## Packages
- `shared-contexts`: Shared React contexts
- `shared-services`: Shared services (Firestore, etc.)
- `shared-hooks`: Shared custom hooks
- `shared-components`: Shared UI components
- `shared-utils`: Shared utilities
- `shared-constants`: Shared constants
- `shared-config`: Shared configuration

## Running Apps
```bash
# Workout app
npm run workout

# Calendar app
npm run calendar
```
